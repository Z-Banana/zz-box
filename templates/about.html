🎯 项目介绍

跨设备实用工具箱是一个完全免费、开源的在线工具集合，旨在帮助用户在不同设备之间实现便捷的文件传输、信息共享和互动协作。

无需下载安装任何软件，只需打开浏览器即可使用。所有功能均基于 Web 技术构建，支持手机、电脑、平板等各类设备。

✨ 主要功能

  • 📁 点对点文件传输 — 基于 WebRTC 技术，文件不经过服务器，直接设备间传输
  • 🔗 网站快捷分享 — 生成二维码或一键发送到其他设备
  • 📋 剪贴板同步 — 跨设备复制粘贴，实时同步
  • 🎮 手机遥控 — 用手机控制电脑上的 PPT/视频播放
  • ⏰ 共享时钟 — 多人同步计时器和闹钟
  • 💰 AA 账单 — 多人消费自动分摊计算
  • 🎲 随机工具 — 掷骰子、随机抽人、投票、决定器

🔒 隐私说明

本项目高度重视用户隐私：

  • 文件传输采用点对点加密，不经过服务器存储
  • 不收集任何个人身份信息
  • 所有数据仅在会话期间临时存储，关闭页面后自动清除
  • 无需注册账号即可使用全部功能

🛠 技术栈

  • 前端：原生 HTML5 / CSS3 / JavaScript
  • 后端：Python + Flask + Flask-SocketIO
  • 实时通信：WebSocket + WebRTC
  • 部署：Pxxl App / 任何支持 Python 的平台
